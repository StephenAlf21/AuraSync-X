Common Questions

Everything you need to know about compatibility, privacy, and features.

AuraSync X is built on modern web technologies (WebGL, Web Audio API) and works best on the latest versions of desktop browsers like Google Chrome, Mozilla Firefox, and Microsoft Edge. While it may function on other browsers, we recommend these for the smoothest 60fps visualization experience.

Yes, AuraSync X is completely free and open-source. It is a passion project designed to provide high-quality tools to the community without paywalls. If you enjoy the app, please consider a small donation to help cover server costs.

We support most modern web-compatible formats:

  • Audio: MP3, WAV, OGG
  • Video: MP4, WebM

Both applications also support direct streaming from URLs, provided they link to a compatible raw media stream.

Absolutely. AuraSync X is a client-side application. This means your files are processed locally in your browser's memory and are never uploaded to our servers. We track zero personal data.

The best way to report issues is via our GitHub Issues page. This allows us to track bugs transparently. You can also email us directly via the Contact page.
